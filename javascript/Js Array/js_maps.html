Js maps

A Map holds key-value pairs where the keys can be any datatype. A Map remembers the original insertion order of the keys.

In conclusion, the forEach() and map() methods are both used to loop through arrays and objects. The forEach() method does not return a new array, whereas the map() method returns a new array.
